# Presentation Assets

This folder contains the one-slide, two-minute pitch assets for the Q-H8R
hackathon presentation.

## Files

- `one_slide.html` - self-contained presentation slide. Open this in a browser
  and present full-screen.
- `one_slide.html.png` - rendered preview of the slide.
- `speaker_notes.md` - two-minute script, 20-second backup, Q&A, and demo
  checklist.
- `canva_template.pdf` - source Canva template supplied for the visual style.
- `canva_template_reference.png` - rendered reference image from the Canva
  template.
- `denario_qitt_paper.pdf` - local copy of the OpenReview paper PDF.
- `openreview_reviews.json` - local OpenReview API response for the submission,
  decision, reviews, and checks.
- `openreview_review_summary.md` - concise review-derived demo notes.

## Demo Flow

1. Paste or pre-fill the selected hypothesis from Boris's Denario paper.
2. Start the live demo.
3. Switch to `one_slide.html` while the demo runs in the background.
4. Deliver the two-minute pitch from `speaker_notes.md`.
5. Switch back to the completed demo for the final recommendation.
